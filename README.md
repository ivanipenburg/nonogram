# Nonogram

A terminal-based Nonogram game in implemented in Python. The plan is to implement a Nonogram solver as well.

# How to run

Python 3.9+ is required.

```
cd game
python main.py -f [PUZZLE_FILE_NAME]
```

# Puzzle files

A puzzle file for the puzzle

```
                3  2
       1  2  2  1  2
   3
   4
2  1
   1
   2
```

looks like this:

```
3/4/2,1/1/2
1/2/2/3,1/2,2
```
